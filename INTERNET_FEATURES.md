# 智能体互联网访问功能

## 概述

智能体现已升级，具备了完整的互联网访问能力，可以获取实时信息。

## 新增工具

### 1. **fetch_webpage_content** - 获取网页内容
获取指定URL的网页内容并提取主要文本。

**用法示例：**
```
用户: 获取GitHub主页的内容
智能体: 会访问github.com并提取页面内容
```

**特点：**
- 自动清除HTML标签
- 限制内容长度（防止信息过多）
- 处理网络错误优雅

### 2. **search_google** - Google搜索
使用Google搜索获取最新信息。

**用法示例：**
```
用户: 搜索最新的Python新闻
智能体: 会从Google搜索结果中提取相关信息
```

**特点：**
- 返回搜索结果前5条
- 每条结果包含：标题、URL、摘要
- 可获得时效性最强的信息

### 3. **extract_website_info** - 提取网站信息
从指定网站提取内容，支持按关键词过滤。

**用法示例：**
```
用户: 从CNN首页提取关于科技的新闻
智能体: 会获取CNN主页，过滤出与科技相关的内容
```

**用法示例（带关键词）：**
```
用户: 从TechCrunch提取信息，过滤关键词: AI, 机器学习, 大模型
```

**特点：**
- 提取网页标题
- 支持关键词过滤（用逗号分隔）
- 自动清理无关内容

## 原有工具（本地能力）

### 4. **calculator** - 数学计算
执行数学表达式计算。

```
用户: 计算 2 + 3 * 4
智能体: 计算结果: 14
```

### 5. **get_text_length** - 获取文本长度
统计文本的字符数。

```
用户: hello 有几个字符？
智能体: 文本长度: 5 个字符
```

### 6. **reverse_string** - 反转字符串
反转输入的文本。

```
用户: 反转 "hello"
智能体: 反转结果: olleh
```

## 技术实现

### 依赖库
- **requests**: HTTP请求库，用于网络访问
- **BeautifulSoup4**: HTML解析库，用于提取网页内容
- **langchain**: 工具框架，用于集成工具

### 工具集成
所有网络工具都通过 `@tool` 装饰器集成到智能体中，LangGraph会自动调用它们来处理用户请求。

## 注意事项

1. **网络连接**: 所有网络功能需要可靠的网络连接
2. **速率限制**: 某些网站可能对爬虫有限制，建议添加延迟
3. **响应时间**: 网络请求可能需要较长时间（5-30秒）
4. **隐私合规**: 请确保网络爬取行为符合网站的robots.txt和法律规定
